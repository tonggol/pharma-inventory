의약품 재고관리 시스템 완전 가이드

1

시스템 소개

의약품 재고관리 시스템은 병원, 약국, 의료기관에서 의약품의 입고, 출고, 재고 현황을 효율적으로 관리할 수 있도록 설계된 웹 기반 시스템입니다.

주요 기능

  • 의약품 정보 관리
    의약품 등록, 수정, 삭제 및 상세 정보 관리
  • 실시간 재고 현황
    현재 재고량, 최소 재고 기준, 유효기간 추적
  • 입출고 거래 관리
    입고, 출고, 조정, 폐기 등 모든 거래 기록 관리
  • 종합 보고서
    재고 현황, 유효기간, 거래 내역 등 다양한 보고서 생성
팁
처음 사용하시는 경우 대시보드에서 전체적인 시스템 현황을 먼저 확인해보세요. 각 메뉴별 주요 기능을 파악하는데 도움이 됩니다.
2

로그인 방법

시스템에 접속하여 로그인하는 방법을 안내합니다.

  • 로그인 페이지 접속
    웹 브라우저에서 시스템 URL에 접속합니다.
  • 사용자 정보 입력
    사용자명과 비밀번호를 입력합니다.
  • 로그인 버튼 클릭
    '로그인' 버튼을 클릭하여 시스템에 접속합니다.

주의사항
보안을 위해 로그인 정보를 다른 사람과 공유하지 마세요. 사용 완료 후에는 반드시 로그아웃하시기 바랍니다.

대시보드 둘러보기

로그인 후 가장 먼저 보게 되는 대시보드의 각 구성 요소를 설명합니다.

주요 통계 카드

대시보드 상단에는 다음과 같은 주요 통계 정보가 표시됩니다:

  • 총 의약품 수
    현재 시스템에 등록된 전체 의약품 종류 수
  • 총 재고량
    모든 의약품의 현재 총 재고 수량
  • 재고 부족
    최소 재고 기준 이하로 떨어진 의약품 수
  • 만료 임박
    유효기간이 임박한 의약품 수

의약품 관리

의약품 정보를 등록, 수정, 삭제하는 방법을 설명합니다.

의약품 목록 조회

  • 메뉴 접근
    좌측 메뉴에서 '의약품 관리' > '의약품 목록'을 클릭합니다.
  • 검색 및 필터
    필요시 검색창이나 필터를 사용해 원하는 의약품을 찾습니다.
  • 상세 정보 확인
    의약품명을 클릭하여 상세 정보를 확인할 수 있습니다.
9

자주 묻는 질문

사용자들이 자주 묻는 질문과 답변을 정리했습니다.

비밀번호를 잊어버렸을 때 어떻게 해야 하나요?
로그인 페이지에서 '비밀번호 찾기' 링크를 클릭하여 이메일로 재설정 링크를 받으실 수 있습니다. 또는 시스템 관리자에게 문의하여 비밀번호를 재설정 받으실 수 있습니다.
의약품 재고가 부족할 때 자동으로 알림을 받을 수 있나요?
네, 가능합니다. 시스템 설정에서 알림 기능을 활성화하면 재고가 최소 기준 이하로 떨어질 때 이메일 알림을 받으실 수 있습니다. 설정 방법은 '시스템 설정' 섹션을 참고하세요.
데이터를 Excel로 내보낼 수 있나요?
네, 대부분의 목록 페이지와 보고서에서 Excel, PDF, CSV 형식으로 데이터를 내보낼 수 있습니다. 각 페이지의 '내보내기' 버튼을 클릭하여 원하는 형식을 선택하세요.
잘못 입력한 거래 내역을 수정할 수 있나요?
거래 내역은 수정할 수 없습니다. 대신 조정 거래를 생성하여 오류를 수정할 수 있습니다. 거래 목록에서 해당 거래를 선택한 후 '조정' 버튼을 사용하세요.
시스템 백업은 언제 이루어지나요?
시스템은 매일 자동으로 백업됩니다. 백업 시간과 주기는 시스템 설정에서 확인하실 수 있으며, 필요시 수동으로 백업을 생성할 수도 있습니다.
